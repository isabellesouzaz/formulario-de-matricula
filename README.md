# Estrelas do Amanhã - Formulário de Matrícula

## Sobre o projeto
Projeto de formulário de matrícula para a escola de educação infantil **Estrelas do Amanhã**. A página reúne campos para cadastro da criança, endereço residencial, dados do responsável, opções de turno, escolha de esporte, envio de certidão de nascimento e aceite dos termos. Este projeto foi desenvolvido como parte da **formação Full Stack da Rocketseat**.

![Preview do projeto](/assets/screenshot.png)

## Funcionalidades

- Formulário completo para matrícula infantil.
- Campos para informações pessoais, data de nascimento, sexo e informações médicas.
- Área de upload para certidão de nascimento.
- Seção de endereço residencial com campos de CEP, rua, número, cidade e estado.
- Cadastro de responsável com nome, telefone e e-mail.
- Validação visual para campo de e-mail obrigatório.
- Seleção de turno de estudo.
- Seleção de esporte entre futebol, basquete, natação, yoga, vôlei e boxe.
- Checkbox para aceite de termos e política de privacidade.
- Botões para salvar respostas e finalizar matrícula.
- Layout dividido entre formulário e área visual institucional.

## Como executar

Basta abrir o arquivo `index.html` no navegador. Não há dependências ou instalação necessárias.

```bash
git clone https://github.com/isabellesouzsv/formulario-de-matricula
cd formulario-de-matricula
```

Depois, abra o arquivo:

```bash
index.html
```
